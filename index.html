Tableau Analytics Project (Customer Churn Case Study)

Purpose

This project demonstrates my ability to clean, analyse and visualise complex datasets using Tableau. It simulates a customer churn analysis for a telecom provider, showcasing segmentation, trend analysis and demographic insights.

Tech Stack

ExcelPower QueryTableau

Data Analytics Concepts

SegmentationTrend AnalysisCustomer RetentionDemographic AnalysisData Cleaning

4 Step Process / Summary

  1. Cleaned data in Power Query; removed out-of-range/missing values; created churn flags.
  2. Explored variables and kept those with signal: demographics, contract type, tenure length, city.
  3. Segmented and analysed churn trends to surface high-risk groups and leakage points.
  4. Built an interactive Tableau dashboard with drill-downs by city, contract type, and demography.
